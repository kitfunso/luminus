# Luminus v0.3.0

This is the first unified npm and Python SDK release since the GIS shortlist tranche landed on `master`.

## Highlights

- published `luminus-mcp@0.3.0`
- published `luminus-py==0.3.0`
- added `shortlist_bess_sites` for GB BESS ranking with GIS, revenue, queue, and SSEN DNO context
- added `get_distribution_headroom` for SSEN public DNO headroom lookup
- upgraded `get_grid_connection_intelligence` to use NESO polygon containment first, with nearest-point fallback
- extended the Python SDK with typed helpers for distribution headroom and connection intelligence
- updated the GIS siting and BESS shortlist notebooks around the new helper surface

## Published artifacts

- npm: `luminus-mcp@0.3.0`
- PyPI: `luminus-py==0.3.0`
- git tag: `v0.3.0`

## Install

```bash
npm install -g luminus-mcp@0.3.0
pip install "luminus-py[notebook]==0.3.0"
```
