# v0.4.0 Release Notes

Released: 2026-04-05

## Summary

Full GB DNO coverage, BESS-aware scoring, and a major Python SDK expansion.

## DNO Coverage

All 5 GB DNOs with public headroom data are now supported:

| DNO | Dataset | Auth | Coordinates |
|-----|---------|------|-------------|
| SSEN | Headroom Dashboard CSV | Free, no key | Yes |
| Northern Powergrid | Heat Map Substation Areas | Free, no key | Yes |
| UKPN | DFES Network Scenario Headroom | Free key | Yes |
| SP Energy Networks | NDP NSHR Workbook | Free key | No (alphabetical) |
| ENWL | PRY Heatmap | Free key | Yes |

ENWL uses the `enwl-pry-heatmap` dataset (367 primary substations, monthly refresh). Headroom is reported as firm demand and generation capacity with separate inverter/synchronous/battery splits.

ENWL was also added to `get_embedded_capacity_register` (563 records, >=1MW, with coordinates) and to the parallel headroom query in `get_grid_connection_intelligence`.

## BESS-Specific Scoring

`compare_sites` now accepts an optional `technology` parameter:

| Technology | Verdict | Solar | Grid | Terrain |
|-----------|---------|-------|------|---------|
| solar (default) | 40% | 30% | 20% | 10% |
| bess | 25% | 5% | 45% | 25% |
| hybrid | 30% | 20% | 35% | 15% |

For BESS, grid proximity dominates (cable cost is the biggest capex variable), solar irradiance is nearly irrelevant, and terrain matters more (flat land for container placement).

`shortlist_bess_sites` now queries SSEN, NPG, and UKPN headroom in parallel and returns the closest match, instead of only querying SSEN.

## Python SDK

24 new typed helpers added to `luminus-py`, covering:

- **DNO/GIS tools**: ECR entries/snapshot, flex dispatches/snapshot, constraint breaches, SPEN grid intelligence (queue/DG/curtailment), UKPN grid overview (GSPs/flex zones/faults)
- **Trading tools**: balancing prices, intraday prices, imbalance prices, spread analysis, ancillary prices
- **GIS tools**: NGED connection signal, terrain analysis, land constraints, BESS shortlist, GIS source verification

9 new snapshot models with `from_dict()` classmethods. Total typed helpers: 36 (up from 12).
