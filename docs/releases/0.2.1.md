# luminus-py v0.2.1

This was a Python-SDK-focused follow-up to `0.2.0`.

The npm MCP package remained `luminus-mcp@0.2.0`.

## Highlights

- published `luminus-py` to PyPI via GitHub Trusted Publishing
- added notebook-first helpers for outages, cross-border flows, grid proximity, grid connection queue, and site revenue
- added opt-in typed Python result models for common GIS and revenue workflows
- improved Python-side error translation for startup, configuration, and upstream failures
- added analyst-ready notebook demos for trader, GIS siting, and BESS shortlist workflows
- added CI coverage for Python tests and packaging checks

## Published artifacts

- PyPI: `luminus-py==0.2.1`
- npm package remained `luminus-mcp@0.2.0`
- git tag: `v0.2.1`

## Install

```bash
pip install "luminus-py[notebook]==0.2.1"
```
